** APITOR Connectors: ** This repository contains standalone, zero-dependency telemetry interceptors for **Spring Boot** and **Express.js**. 

Drop these lightweight files into your project to start tracking your core API metrics instantly.
Follow the instructions in files and set up the APITOR_PROJECT_TOKEN envrironment variablse and start tracking your API
